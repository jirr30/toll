# Termux Login System

A structured and modular login system for Termux written in Python.

## Features
- User authentication with SHA-256 hashing
- Admin and user level permissions
- Activity logging
- Menu system with:
  - System information
  - File operations
  - Network tools
  - Package manager
  - User management (admin only)

## Installation
1. Save the files to your Termux
2. Make executable:
   ```bash
   chmod +x login_system.py
